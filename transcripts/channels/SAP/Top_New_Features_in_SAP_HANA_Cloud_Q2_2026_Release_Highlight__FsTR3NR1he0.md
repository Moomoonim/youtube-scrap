## 스크립트

안녕하세요 여러분. 안녕하세요, SAP HANA 클라우드 담당 수석 제품 관리자 토마스입니다. 2026년 두 번째 릴리스가 빠르게 다가오고 있으며, [음악] SAP HANA Cloud에서 곧 출시될 기능에 대한 독점적인 첫 번째 미리보기를 여러분께 제공하게 되어 매우 기쁩니다. 이번 릴리스를 통해 당사의 지능형 AI 기반 엔터프라이즈급 데이터 플랫폼 [음악]이 한 단계 더 발전했습니다. 저희는 데이터 검색 및 쿼리를 더욱 쉽게 만들고, 데이터베이스에서 직접 AI를 활용하며, 음악 그래프 지식 의 가치를 극대화하고 , 데이터베이스 관리를 간소화하는 혁신적인 기능을 도입하고 있습니다. 그럼 바로 본론으로 들어가서 여러분을 위해 준비한 주요 내용들을 살펴보겠습니다. 먼저 SAP HANA 클라우드에서 에이전트 방식의 [음악] 데이터 접근을 가능하게 하는 새로운 혁신 기술에 대해 이야기해 보겠습니다 . 모든 것은 [음악] 사용자 지정 데이터베이스 객체 지식 그래프를 자동으로 생성하는 새로운 기능에서 시작됩니다. SAP HANA Cloud는 테이블, 뷰, 열, 관계를 포함한 데이터베이스 메타데이터를 스캔하여 데이터 환경의 의미론적 표현을 자동으로 구축할 수 있습니다 . 이 [음악]은 수동 의미 모델링의 필요성을 없애고 데이터 검색, 추론, [음악] 및 AI 기반 상호 작용을 위한 풍부한 기반을 제공합니다. 이러한 기반 위에 데이터베이스 객체 검색 도구와 데이터 검색 도구를 새롭게 선보일 예정이며, 이 도구들은 7월에 출시될 것입니다. 오늘날 사용자들은 데이터가 어디에 저장되는지, 데이터베이스 객체들이 어떻게 연결되어 있는지, 그리고 정보를 검색하기 위해 SQL 쿼리를 어떻게 작성하는지 알아야 하는 경우가 많습니다 . 이러한 새로운 기능을 사용하면 자연어를 사용하여 원하는 것을 간단하게 설명할 수 있습니다. 데이터베이스 객체 검색 도구는 지식 그래프를 사용하여 관련 데이터베이스 객체와 관계를 식별하고 , 데이터 검색 도구는 요청을 SQL로 변환하여 SAP HANA 클라우드에서 직접 실행합니다. 그 결과, 데이터베이스 스키마나 SQL에 대한 깊은 이해 없이도 기업 데이터를 탐색, 이해, 쿼리하는 훨씬 간단한 방법을 제공하여 음악 관련 AI 시나리오를 구현할 수 있게 되었습니다. 이러한 혁신 기술들은 SAP HANA 클라우드의 멀티 모델 기반과 음악 지식 그래프 엔진을 활용하여 차세대 대화형 및 에이전트 기반 음악 데이터 경험을 구현합니다. 다음으로, 지식 그래프라는 주제를 계속 살펴보겠습니다. 이번 릴리스를 통해 SAP HANA Cloud는 지식 그래프를 속성 그래프로 변환하는 기능을 지원합니다. 이를 통해 의미 검색, 음악 그래프 탐색 또는 고급 그래프 분석 등 시나리오에 가장 적합한 그래프 표현 방식을 유연하게 사용할 수 있습니다. 또한 서비스 명세를 통해 SPARQL 페더레이션 쿼리 지원을 도입할 예정입니다 . 이를 통해 데이터를 이동하거나 사용자 지정 통합 레이어를 구축할 필요 없이 단일 쿼리를 통해 여러 SAP HANA Cloud 환경에 분산된 지식 그래프를 쿼리할 수 있습니다. 그 결과 분산된 지식에 대한 접근성이 향상되고 AI, 의미 검색, 검색 증강 생성 및 음악 추론 워크로드에 대한 더욱 견고한 기반이 마련됩니다. 이번 릴리스의 또 다른 주요 특징은 AWS에서 더 큰 규모의 SAP HANA 클라우드 데이터베이스 인스턴스를 지원한다는 점입니다. SAP HANA 클라우드는 이제 최대 24TB의 메인 메모리를 갖춘 데이터베이스 인스턴스를 제공하여 기업이 음악 관련 작업 부하를 처리하는 데 필요한 용량을 더욱 확대할 수 있도록 지원합니다. 이러한 인스턴스 크기는 전용 예약 인프라에서 실행되어 미션 크리티컬 애플리케이션 및 대규모 분석 워크로드에 필요한 리소스에 대한 보장된 액세스를 제공합니다. 이를 통해 SAP 클라우드는 완전 관리형 클라우드 서비스의 간편함을 유지하면서도 가장 까다로운 엔터프라이즈 시나리오까지 지원할 수 있는 역량을 더욱 확장합니다 . SAP HANA 클라우드와 SAP AI 코어 간의 더욱 심층적인 통합을 계속해서 살펴보겠습니다. SAP HANA Cloud 내에서 SAP AI Core의 테이블 형식 AI 모델 서비스를 직접 사용할 수 있도록 하는 새로운 SQL 예측 함수를 소개합니다. 즉, 자체 머신러닝 모델을 만들거나, 학습시키거나, 관리할 필요 없이 바로 사용할 수 있는 AI 모델을 활용하여 구조화된 비즈니스 데이터에 대한 예측을 생성할 수 있다는 의미입니다 . SQL 프로시저를 통해 SAP 파운데이션 모델과 SAP ABAP 모델에 직접 액세스할 수 있으므로 AI 기반 음악 예측 기능을 애플리케이션 및 비즈니스 프로세스에 통합하는 데 필요한 노력을 크게 줄일 수 있습니다. 그게 다가 아닙니다. 또한 SAP AI Core를 통해 대규모 언어 모델에 액세스할 수 있는 네이티브 SQL 함수인 AI 텍스트 완성 기능을 새롭게 선보입니다 . 이를 통해 LLM 기반 [음악] 텍스트 생성 기능을 데이터베이스 워크플로 및 SQL 분석에 직접 통합하는 동시에 SAP AI Core를 안전하고 관리되는 엔터프라이즈급 AI 서비스 게이트웨이로 활용할 수 있습니다. 이러한 혁신들을 통해 AI [음악] 기능이 데이터에 더욱 가까워지고 SAP HANA 클라우드에서 지능형 애플리케이션을 구축하는 것이 그 어느 때보다 쉬워집니다 . 마지막으로, 2026년 6월 출시 예정인 새로운 기능들을 간략하게 살펴보겠습니다. 여기에는 리소스 활용도를 지속적으로 분석하고 과거 작업 부하 패턴을 기반으로 가장 적합한 데이터베이스 크기 구성과 코어-메모리 비율을 자동으로 추천하는 새로운 성능 클래스 어드바이저인 [music]이 포함됩니다. 또한 SQL Plan Advisor에 자동화된 실행 계획 검증 및 최적화 기능을 추가하여 수동 개입 없이 최적의 [음악] 쿼리 성능을 복원할 수 있도록 지원하고 있습니다 . Lakehouse 시나리오의 경우, SAP HANA Cloud는 이제 Apache Iceberg REST 카탈로그를 지원하여 주요 클라우드 제공업체의 최신 엔터프라이즈 Iceberg 환경과 직접 통합할 수 있습니다 . 또한 SAP S/4 HANA CDS View 엔터티에 대한 복제 지원을 도입하여 데이터를 SAP HANA 클라우드로 직접 복제함으로써 소스 시스템에 미치는 영향을 줄이고 빠른 로컬 분석을 수행할 수 있도록 합니다 . 늘 그렇듯이, 이것들은 저희 2분기 업데이트에 포함될 수많은 혁신 기능 중 일부에 불과합니다 . 이러한 내용에 대해 더 자세히 알아보려면 곧 개최될 SAP HANA 클라우드 신규 기능 웹캐스트를 시청하거나 당사 웹사이트를 방문하십시오. 시청해 주셔서 감사합니다. 최신 소식을 받아보시려면 구독 버튼을 눌러주세요. 곧 다시 만나요.
