## 스크립트

인공지능은 그 어느 때보다 많은 코드를 생성하고 있으며, 위협 행위자들은 인공지능을 이용하여 소프트웨어 취약점을 악용하고 있습니다. 이 새로운 경쟁 환경에서 보안 및 개발 팀은 '왼쪽으로' 이동해야 합니다. Google DeepMind가 개발한 AI 기반 코드 보안 에이전트인 Code Minder를 만나보세요. 이제 Gemini Enterprise Agent Platform을 통해 Google Cloud 고객이 이용할 수 있습니다 . Code Minder는 Gemini 또는 기타 파운데이션 모델을 사용하여 코드베이스를 검사하고 , 심각한 취약점을 찾아내고 수정합니다 . 어떻게 작동하는지 살펴보겠습니다. Code Minder는 소스 코드 저장소에 직접 연결되는 VS Code와 같은 개발자 도구와 연동됩니다 . Code Minder는 Gemini와 같은 고급 모델을 사용하여 코드베이스의 심각한 취약점을 검사합니다. 검사가 완료되면 심각도에 따라 우선 순위가 지정된 보안 발견 사항 보고서가 제공됩니다 . Code Minder는 이러한 결과를 형식화된 보고서로 생성하여 이해 관계자들과 공유할 수도 있습니다. 취약점을 발견한 후, Code Minder는 검증 계획을 생성하고 개념 증명 익스플로잇을 실행하여 각 취약점이 실제로 위험 요소인지 확인합니다. 여기에서 보시는 바와 같이 Code Minder가 해당 발견 사항을 성공적으로 검증했습니다. Code Minder는 이제 패치를 생성하고 테스트하여 해당 취약점을 해결할 수 있습니다 . 여기서 Code Minder가 패치를 성공적으로 생성한 것을 확인할 수 있습니다. 개발자는 에이전트가 수행한 코드 변경 사항을 검토하고 어떤 변경 사항을 코드베이스에 반영할지 결정하여 완전한 제어권을 유지할 수 있습니다. 정말 간단하죠. 개발자를 위한 보안 도우미인 Code Minder를 사용하면 보안을 초기 단계로 옮기는 것이 훨씬 쉬워집니다. 소프트웨어의 심각한 취약점을 제품 출시 전에 스캔, 검증 및 수정하고, 동시에 개발자에게 진행 상황을 지속적으로 알립니다 . Wiz 및 Mandiant [music] Consulting과 함께 Code Minder는 상시 가동되는 자율 보안 플랫폼인 Google AI 위협 방어의 핵심 요소입니다 . 구글 AI 위협 방어로 AI 시대를 지켜내세요 .
