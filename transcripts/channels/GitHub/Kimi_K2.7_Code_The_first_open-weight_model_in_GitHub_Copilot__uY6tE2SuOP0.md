## 스크립트

오늘 GitHub는 Kimmy K 2.7 코드 모델이 GitHub Copilot 모델 라인업에 추가되었다고 발표했습니다. 그렇다면 이 키미 모델이란 무엇이며, 이 모델에 대해 알아야 할 점은 무엇일까요? 한번 살펴볼까요? Kimmy는 Moonshot AI라는 회사에서 만든 모델이며, K 2.7 코드는 코딩에 초점을 맞춘 에이전트 모델의 최신 버전입니다. 그리고 이 모델은 개방형 가중치 모델로 알려져 있습니다. 이는 키미를 훈련시키는 데 사용된 매개변수가 모든 사람에게 공개된다는 것을 의미합니다. 따라서 이 모델은 수정하거나 미세 조정할 수 있습니다. 이로써 Kimmy는 GitHub Copilot 모델 라인업에 추가된 최초의 오픈 웨이트 모델이 되었습니다 . Kimmy 모델은 미국에 기반을 둔 Microsoft AI Foundry 인프라에서 호스팅되며 GitHub와 Microsoft에서 관리합니다. Kimmy K 2.7 코드를 사용하려면 모델 선택기에서 해당 코드를 찾으세요. 참고로, 키미는 기본적으로 비활성화되어 있습니다. 따라서 해당 기능이 보이지 않는다면 조직 또는 관리자가 해당 기능을 활성화하지 않았을 가능성이 높습니다. 이를 확인하려면 Copilot CLI의 모델 탭에서 계정에 대해 활성화되지 않은 모델을 확인할 수 있습니다. Kimmy K 2.7 코드는 100만 토큰당 95 AI 크레딧의 입력 배율을 가지고 있습니다. 그리고 AI 크레딧 1점은 대략 1센트와 같다는 점을 기억하세요. 그러니까 입력 토큰 100만 개당 95센트가 되는 셈입니다 . 또한 출력 토큰 100만 개당 AI 크레딧 400개가 지급됩니다 . 이로써 해당 모델은 GitHub Copilot에서 가장 저렴한 모델 중 하나가 되었습니다. 내부 테스트 결과, Kimmy K 2.7 코드는 최첨단 에이전트 코딩 모델의 정확도 및 기능에 필적하는 것으로 나타났습니다. 하지만, 이 제품은 다양한 작업에도 매우 적합합니다 . 또한 첫 토큰 획득 시간과 완료 시간 면에서 라인업 중 가장 빠른 축에 속합니다. 또한 95%의 현금 적중률을 자랑하며, 라인업에서 가장 효율적인 카드 중 하나입니다 . 예를 들어, 링크 공유 사이트에 대한 전체 프로젝트 요구사항 문서의 구현 계획을 세우라는 요청을 받았을 때 , Kimmy K2.7 코드는 가정을 파악하고 세부 사항에 대한 높은 집중력을 보여주는 8가지 질문을 제시할 수 있었습니다 . 키미는 해당 계획을 자동화 모드로 실행하면서 모든 작업을 완료할 수 있었고, 최소한의 후속 조치만으로도 정상적으로 작동하는 애플리케이션을 제공할 수 있었습니다. Kimmy K2.7 코드는 오늘부터 모든 프로, 프로 플러스 및 맥스 사용자에게 제공되며, 비즈니스 및 엔터프라이즈 고객은 향후 몇 주 내에 이용할 수 있게 될 예정입니다. 저희는 모델 선택이 개발자에게 매우 중요하다는 것을 알고 있으며, 성능과 가격 면에서 더 나은 모델 선택권을 제공하기 위해 GitHub Copilot에 새로운 모델을 지속적으로 평가하고 추가할 예정입니다. 즐거운 코딩 되세요.
