## 스크립트

[음악] Azure Migrate 시리즈의 새로운 에피소드에 오신 것을 환영합니다. [음악] 이 비디오에서는 Azure VM 대상에 대한 Azure Migrate 평가에 대해 설명합니다. 평가의 중요성과 활용법을 다룬 이전 영상들을 시청하고, 평가에 필요한 사전 요건을 모두 충족한 후에 다음 단계로 진행하는 것이 좋습니다. Azure Migrate 프로젝트를 생성하고 Windows 및 Linux 서버의 인벤토리를 검색했는지 확인하십시오. Azure Migrate 어플라이언스를 사용하거나 RVTools 또는 CMDB에서 캡처한 인벤토리를 가져와 인벤토리를 검색할 수 있습니다. 검색을 위해 장치를 사용한 경우, 더 나은 평가 결과를 얻기 위해 성능 데이터를 축적하려면 최소 24시간 동안 검색을 실행하는 것이 좋습니다 . 이제 Azure 포털에서 Azure VM 평가 데모를 바로 살펴보겠습니다. Azure Migrate를 검색하고 프로젝트로 이동하세요 . 왼쪽에는 ' 인벤토리 탐색' 아래에 검색된 모든 Windows 및 Linux 서버의 인벤토리가 포함된 인프라 블레이드가 표시됩니다. 평가를 생성하려면 몇 개의 가상 머신을 선택해야 합니다. 특정 열 속성을 사용하거나 태그를 사용하여 필터를 적용할 수 있습니다. 몇몇 기기를 선택하고 ' 평가 생성'을 클릭하세요. Azure VM 대상에 대한 평가를 생성하려면 드롭다운 메뉴에서 Azure VM을 선택하세요. 다음으로, 요구 사항에 맞게 평가 속성을 사용자 지정할 수 있습니다. 대상 위치에 따라 평가에 사용할 Azure 지역이 결정됩니다 . 지역별로 SKU의 가격과 재고 상황이 다르기 때문에 이는 중요합니다. 또한 개발, 테스트 또는 프로덕션과 같은 환경 및 소스 워크로드 유형을 지정하여 적절한 가격을 적용할 수 있습니다. 종량제, 1년 또는 3년 예약 인스턴스, Azure 절약 계획 및 평가 기준과 같은 통화 절약 옵션을 통해 최대 절감 효과를 얻으면서 총소유비용(TCO)을 예측하고 VM 대상에 가장 적합한 규모의 인스턴스를 식별할 수 있습니다. ' 기본값 편집'을 클릭하면 대상 VM, SKU, 디스크 유형 등과 같은 Azure VM 대상별 속성을 지정할 수 있습니다. 모든 세부 정보를 지정한 후 ' 검토 및 평가 생성'을 클릭하세요 . 모든 것이 괜찮아 보이면 ' 만들기'를 클릭하세요. 평가 결과는 몇 분 안에 검토 가능합니다. 이 평가를 다시 검토해 봅시다. 여기에서 보시는 바와 같이, 저희는 총 8개의 서버를 평가했으며, 그중 6개는 준비 완료 상태 이고 2개는 조건부로 준비 완료 상태로 표시되어 있습니다. 개요에는 예상 월별 비용에 대한 세부 정보도 제공되며, 이 경우 비용은 1,176달러입니다. 좀 더 자세히 살펴보겠습니다. 보시는 바와 같이 5개의 서버가 지원 종료로 표시되어 있습니다. 지원 상태 및 운영 체제별 배포 현황은 ' 지원 세부 정보 보기'를 클릭하여 확인할 수 있습니다. 이제 각 VM에 해당하는 마이그레이션 권장 사항을 살펴보겠습니다 . AirSonic MySQL을 예로 들어보겠습니다. 요약 탭에서는 준비 대상 구성, 권장 마이그레이션 도구 및 이 VM에 적용된 태그에 대한 간략한 세부 정보를 확인할 수 있습니다. 대상 구성을 식별하는 데 사용된 소스 속성을 검토할 수 있습니다 . 여기서 보시는 바와 같이, 해당 VM은 80GB 디스크와 2개의 IOP 요구 사항을 가지고 있었고, 메모리 사용률은 7.99%였습니다 . 대상 권장 사항 탭에서는 컴퓨팅, 운영 체제 및 스토리지 속성을 포함하여 대상 VM 구성에 대한 모든 세부 정보를 확인할 수 있습니다. 각 영역별 월별 비용 내역을 시각화할 수도 있습니다. 평가 결과가 어떻게 산출되었는지 이해하기 위해 권장 사항의 근거를 검토할 수도 있습니다. 탐색 및 평가가 완료되고 신뢰도 높은 마이그레이션 계획을 수립한 후에는 Azure Migrate의 기본 서버 마이그레이션 기능을 활용하여 서버 배포를 마이그레이션할 수 있습니다 . 서버를 Azure 가상 머신으로 마이그레이션할 수 있습니다 . 실제 마이그레이션을 진행하기 전에 프로덕션 워크로드에 영향을 주지 않고 테스트 마이그레이션을 수행할 수도 있습니다 . [음악] 이 영상을 시청해 주셔서 감사합니다.
