## 스크립트

친애하는 젯브레인즈 팬 여러분, 여러분께 아주 좋은 소식이 있습니다. 이제 AI 어시스턴트를 통해 즐겨 사용하는 IDE에 GitHub Copilot을 기본적으로 추가할 수 있습니다. JetBrains의 새로운 에이전트 클라이언트 프로토콜 지원으로 AI 에이전트 통합이 매우 간편해졌습니다. GitHub Copilot을 단 몇 번의 클릭만으로 실행하는 방법을 보여드리겠습니다. 먼저, 저처럼 IntelliJ에서 JetBrains IDE를 열고 AI 에이전트 탭으로 이동하세요 . 목록에서 GitHub Copilot을 선택하세요 . 그게 다예요. 이제 JetBrains IDE의 익숙한 채팅 창을 통해 GitHub Copilot을 사용할 수 있습니다 . 이 프로젝트가 무엇에 관한 것인지 간략하게 설명해 주세요. 아니면, README 파일이 최신 버전인가요? 또는 좀 더 복잡한 요청, 예를 들어 " 테스트를 실행하고 테스트 범위가 충분한지 알려줘"와 같은 요청도 가능합니다. 계속 지켜봐 주세요. 향후 개선 사항에는 다음 편집 제안, 코파일럿 스킬, 복잡한 개발 작업을 위한 더욱 심층적인 오케스트레이션 기능이 포함될 예정입니다 . 이는 JetBrains와 GitHub 간의 지속적인 협력으로, 통합을 더욱 강화하고 사용자 경험을 개선하는 것을 목표로 합니다.
