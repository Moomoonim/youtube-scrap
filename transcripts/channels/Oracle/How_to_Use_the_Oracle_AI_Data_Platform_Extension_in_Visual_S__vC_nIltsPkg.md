## 스크립트

이 튜토리얼에서는 Visual Studio Code에서 Oracle AI 데이터 플랫폼 개발자 확장 프로그램을 사용하는 방법을 살펴보겠습니다 . 이 확장 프로그램은 Visual Studio Code 내에서 Oracle AI 데이터 플랫폼 리소스에 직접 액세스할 수 있도록 해줍니다 . 연결 후에는 개발 환경을 벗어나지 않고도 에이전트, 워크플로, 지식 기반 및 작업 공간 파일을 사용할 수 있습니다 . 이 시나리오에서는 확장 프로그램을 설치하고 Oracle AI 데이터 플랫폼 인스턴스에 연결한 다음 해당 기능을 살펴보려고 합니다 . 시작하려면 먼저 Visual Studio Code Marketplace를 열고 Oracle AI 데이터 플랫폼 개발자 확장 프로그램을 검색하겠습니다. Visual Studio Code에 확장 프로그램을 추가하려면 설치를 선택하겠습니다. 설치가 완료되면 Visual Studio Code 사이드바에서 확장 프로그램을 사용할 수 있게 됩니다. 확장 프로그램이 설치되었는지 확인한 다음, 확장 프로그램을 열고 브라우저 로그인을 사용하여 인증하겠습니다 . 다음으로 Oracle AI 데이터 플랫폼 URL을 입력하겠습니다. 팝업 창이 나타나면 내 계정 정보를 사용하여 로그인하겠습니다. 이제 확장 프로그램은 AI 데이터 플랫폼의 정보를 표시합니다. 워크스페이스를 선택하면 해당 워크스페이스 에서 사용 가능한 파일과 리소스에 접근할 수 있습니다 . Visual Studio Code에서 에이전트 리소스를 직접 찾아보겠습니다. 에이전트는 시각적 개발 환경과 코드 기반 개발 환경 모두를 지원합니다. 이 확장 프로그램을 통해 코드 기반 에이전트를 편집, 생성 및 관리할 수 있지만, 시각적 흐름을 통해 생성된 에이전트는 관리만 가능합니다. 먼저 에이전트를 정의하는 파일을 살펴보겠습니다 . 파일 내용을 확인하고 싶으면 마우스 오른쪽 버튼을 클릭하고 '파일 미리보기'를 선택하면 됩니다 . 파일을 편집하려면 마우스 오른쪽 버튼을 클릭하고 ' 파일 다운로드'를 선택하겠습니다. 변경 사항을 적용한 후에는 다시 마우스 오른쪽 버튼을 클릭하고 '서버로 푸시'를 선택하여 변경 사항을 저장합니다. 또한, 제가 저장하는 파일들이 수동으로 업로드하고 저장하는 대신 자동으로 AIDP 인스턴스에 업로드되고 저장되도록 설정을 변경할 수도 있습니다 . 다음으로, 상담원이 수행할 수 있는 작업들을 살펴보겠습니다 . 저는 컴퓨팅 리소스 연결이나 배포와 같은 작업을 수행하면서 에이전트를 완벽하게 관리할 수 있습니다. 해당 창에서 플레이 아이콘을 선택하여 배포한 다음, 플레이그라운드를 선택하고 배포만 하겠습니다. 팝업 창에서 에이전트에 쿼리를 보내 동작을 테스트하고 에이전트 호출 의 시각적 추적과 같은 정보를 볼 수 있습니다 . 이제 컴퓨팅 리소스와 워크플로에 대해 설명하겠습니다. 이 확장 프로그램을 사용하면 인터페이스로 돌아가지 않고도 두 가지 모두 검토할 수 있습니다. 워크플로 실행을 선택하면 Visual Studio Code 내에서 직접 상태를 모니터링하고 실행 세부 정보를 확인할 수 있습니다 . 다음으로 마스터 카탈로그를 열겠습니다. 여기에서 스키마, 볼륨, 지식 기반을 포함한 공유 데이터 리소스에 액세스할 수 있습니다. 마지막으로, AIDP 클러스터에서 실행을 선택하여 파이썬 파일을 실행하겠습니다. 대상 클러스터를 선택 하고 실행하겠습니다. 이 확장 프로그램은 선택한 클러스터에서 코드를 실행 하고 실행 결과를 Visual Studio Code 내에 직접 반환합니다. 이제 Oracle AIDP 개발자 확장 프로그램을 설치하고 연결했으므로 Visual Studio Code에서 Oracle AI 데이터 플랫폼 리소스에 직접 액세스하고 관리할 수 있습니다 . 아래 링크에서 이 시리즈의 다른 동영상과 Oracle AI 데이터 플랫폼 관련 추가 자료를 찾아보세요.
